# turno_express_flutter_2024

<h2>Acerca de la aplicacion</h2>
<p>
  La aplicación de gestión de turnos está cuidadosamente diseñada para satisfacer las
necesidades de dos grupos de usuarios clave: los dueños de negocios y los clientes. Para los
primeros, la plataforma ofrece una herramienta integral que permite registrar sus empresas
de manera detallada, describir los servicios que ofrecen, establecer horarios laborales
específicos y administrar la disponibilidad de turnos con facilidad. Los dueños de negocios
pueden configurar perfiles empresariales de manera sencilla y gestionar de forma eficiente
todos los turnos entrantes, lo que les permite optimizar la organización de su tiempo y
recursos. Además, tienen la posibilidad de acceder a un panel de control intuitivo que les
facilita la supervisión y la toma de decisiones estratégicas respecto a la gestión de sus
servicios.
</p>
<p>
  Por otro lado, los clientes disfrutan de una experiencia fluida y conveniente al utilizar
la aplicación. Tienen la posibilidad de explorar un amplio listado de negocios registrados,
verificar en tiempo real la disponibilidad de turnos y realizar reservas de servicios de manera
rápida y sencilla. Una vez realizado el proceso de reserva, los clientes reciben notificaciones
automáticas que les informan sobre el estado de sus turnos, asegurándose de que estén
siempre actualizados. Un punto destacado de la aplicación es su capacidad para proporcionar
actualizaciones en tiempo real sobre el estado de los turnos, permitiendo a los clientes
conocer exactamente cuándo es su turno y cuánto tiempo de espera pueden anticipar, lo que
mejora significativamente la transparencia y la satisfacción del usuario.
</p>
<h2>Arquitectura del proyecto </h2>
<p>Este proyecto fue realizado para dispositivos mobiles y para cualquier dispositivo que pueda navegar por la web. Con la posibilidad de expandir nuestro producto a cualquier interfaz de usuario gracias a la incorporacion de una API. a continuacion las tecnologias que utlizamos para realizar este ambicioso proyecto: </p>
<ul>
  <li>Flutter (Mobile)</li>
  <li>React (Pagina Web)</li>
  <li>ASP.Net core Api (API)</li>
</ul>
